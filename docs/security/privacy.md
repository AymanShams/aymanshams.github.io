# privacy and security

## v1 privacy position

The site should not collect visitor data.

## prohibited in v1

- Forms.
- Cookies.
- Analytics.
- Tracking pixels.
- Third-party scripts.
- API keys.
- Secrets.
- Visitor accounts.
- Patient data.
- Client data.
- Payer data.
- Internal healthcare or insurance operating procedures.

## content review

Before publishing, review for:

- OpenAI affiliation confusion.
- Unsupported claims about Codex Coding OS.
- Sensitive HealPath or SiriusShield detail.
- PHI.
- Client, patient, payer, payment, contract, or internal operations detail.

## link safety

External links that open in a new tab should use:

```html
rel="noopener noreferrer"
```

## later changes

Any future analytics, forms, newsletter, custom scripts, or embedded services require a documentation update before implementation.
