# frontend overview

## stack

- Static HTML.
- Static CSS.
- No JavaScript required for v1.

## pages

| Page | File |
|---|---|
| Homepage | `index.html` |
| Codex Coding OS | `codex-coding-os/index.html` |
| Healthcare operations | `healthcare-operations/index.html` |
| Notes | `notes/index.html` |
| Not found | `404.html` |

## layout rules

- Text-only v1.
- No decorative hero image.
- No popups.
- No nested cards.
- No newsletter box.
- No chat widget.
- No large marketing sections.

## styling rules

- Use one CSS file at `assets/css/styles.css`.
- Use readable text sizes.
- Keep line length comfortable.
- Use strong contrast.
- Ensure navigation wraps cleanly on mobile.
