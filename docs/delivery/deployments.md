# deployments

## v1 deployment target

GitHub Pages from the root of `main` in:

```text
AymanShams/aymanshams.github.io
```

## planned URL

```text
https://aymanshams.github.io/
```

## setup steps after coding is approved

1. Create or clone the approved repository.
2. Add approved static files.
3. Commit and push to `main`.
4. In GitHub, open repository settings.
5. Open Pages settings.
6. Select deploy from branch.
7. Select branch `main`.
8. Select folder root.
9. Save.
10. Wait for the Pages deployment.
11. Open the published URL.
12. Run launch validation.

## rollback

Rollback is a Git revert to the last known good commit, followed by GitHub Pages redeployment.
