# glossary

## ADR

Architecture Decision Record. A short document that records an important technical decision, the alternatives considered, and the consequences.

## GitHub Pages

GitHub's static website hosting service.

## PHI

Protected health information. Health information that can identify a person and must not be published on this site.

## PRD

Product Requirements Document. The controlled source for product scope, users, goals, and release requirements.

## TDD

Technical Design Document. The controlled source for technical structure, implementation slices, and validation rules.

## WBS

Work Breakdown Structure. A breakdown of deliverables, dependencies, validation gates, and sequence.
