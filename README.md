# Ayman Shams personal site

## what this is

Personal GitHub Pages site for Ayman Shams.

Purpose:
- Act as the public umbrella for Ayman's work.
- Route readers to Codex Coding OS, healthcare operations, medical insurance operations, and operating notes.
- Keep the first version lean, serious, and easy to maintain.

Main users:
- Technical reviewers.
- Founders and partners.
- Healthcare operators.
- People checking Ayman's work from GitHub or LinkedIn.

System owner:
- Ayman Shams.

## current status

This project is in documentation setup. Site implementation is blocked until the workflow manifest permits coding.

## planned v1 stack

- Static HTML.
- Static CSS.
- GitHub Pages.
- No backend.
- No database.
- No forms.
- No analytics.
- No custom domain for v1.
